Privacy Policy

Last updated: December 26, 2024

RepoJudge ("we", "our", or "us") is committed to protecting your privacy. This Privacy Policy explains how we collect, use, and safeguard your information when you use our service.

1. Information We Collect

We collect the following types of information:

  • GitHub Account Data: When you authenticate via GitHub OAuth, we receive your public profile information (username, email, avatar) and access to your repositories (public or private, based on your authorization).
  • Repository Data: We temporarily process repository content (code, file structure, README files) to perform analysis. This data is sent to Google's Gemini AI for processing.
  • Usage Data: We may collect anonymous usage statistics to improve our service.

2. How We Use Your Information

  • To authenticate you with GitHub
  • To analyze your code and generate reports
  • To generate README files and recommendations
  • To improve our AI analysis capabilities
Important: We do NOT store your source code permanently. Repository content is processed in-memory and discarded after analysis.

3. Third-Party Services

We use the following third-party services:

  • GitHub API: For authentication and repository access
  • Google Gemini AI: For code analysis and content generation

These services have their own privacy policies. We encourage you to review them.

4. Data Security

We implement industry-standard security measures to protect your data. However, no method of transmission over the Internet is 100% secure.

5. Your Rights

You can:

  • Revoke GitHub access at any time from your GitHub settings
  • Request deletion of your account data by contacting us
  • Access information about what data we have collected

6. Cookies

We use essential cookies for session management. No third-party tracking cookies are used.

7. Changes to This Policy

We may update this policy from time to time. We will notify users of significant changes via the website.

8. Contact Us

If you have questions about this Privacy Policy, please contact us at privacy@repojudge.com.
